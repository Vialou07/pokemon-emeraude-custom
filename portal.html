Comment jouer ?

1. Connexion

Entrez votre pseudo sur la page de jeu. C'est votre identifiant unique pour les sauvegardes cloud.

2. Controles

Sur mobile, les touches tactiles apparaissent automatiquement. Sur PC : fleches + Z X Enter Backspace.

3. Sauvegarder (IMPORTANT)

Menu en jeu > "Sauvegarder". NE FERMEZ PAS la page tant que le message vert "Sauvegarde reussie !" n'est pas apparu. Il n'y a pas de sauvegarde automatique. Si vous fermez avant, votre progression est perdue.

4. Tips

Explorez chaque recoin ! Des Pokemon rares se cachent partout. Battez les dresseurs avec des Fakemons pour les recevoir en cadeau.
