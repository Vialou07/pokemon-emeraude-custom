3 modes de jeu

Au debut de chaque nouvelle partie, choisissez votre mode de difficulte. Ce choix est definitif !

Normal

Le jeu classique, sans restriction. Capturez autant de Pokemon que vous voulez, soignez-les au Centre Pokemon. Ideal pour decouvrir les Fakemons et l'aventure a votre rythme.

Nuzlocke

1 seule capture par zone. Seul le tout premier Pokemon sauvage rencontre dans chaque zone peut etre capture. S'il s'enfuit ou est mis KO, cette zone est perdue pour toujours. Le tracking commence quand vous recevez vos premieres Pokeballs.

Nuzlocke Hardcore

Toutes les regles du Nuzlocke, plus : un Pokemon KO est definitivement perdu. Il est relache automatiquement a la fin du combat. Si toute votre equipe tombe a 0 PV, c'est le Game Over — retour au titre.

Comment jouer ?

1. Connexion

Entrez votre pseudo sur la page de jeu. C'est votre identifiant unique pour les sauvegardes cloud.

2. Controles

Sur mobile, les touches tactiles apparaissent automatiquement. Sur PC : fleches + Z X Enter Backspace.

3. Sauvegarder (IMPORTANT)

Menu en jeu > "Sauvegarder". NE FERMEZ PAS la page tant que le message vert "Sauvegarde reussie !" n'est pas apparu. Il n'y a pas de sauvegarde automatique. Si vous fermez avant, votre progression est perdue.

4. Tips

Explorez chaque recoin ! Des Pokemon rares se cachent partout. Battez les dresseurs avec des Fakemons pour les recevoir en cadeau.

Editeur de Pokemon

Depuis l'onglet Editeur, vous pouvez modifier directement les Pokemon de votre equipe. Voici les regles :

Redistribution des EVs

Redistribuez librement les EVs (Effort Values) de chaque Pokemon. Condition : le Pokemon doit avoir accumule 510 EVs au total avant de pouvoir redistribuer. Si ce n'est pas le cas, faites plus de combats ou utilisez des Baies en jeu. Maximum 252 par stat, 510 au total.

Changement de Nature

Modifiez la nature de vos Pokemon pour optimiser leurs stats. Condition : avoir obtenu les 8 badges d'arene. Chaque nature donne +10% sur une stat et -10% sur une autre (ou neutre). Bleu = +10%, Rouge = -10%.

IVs (valeurs individuelles)

Les IVs sont les talents innees de chaque Pokemon, de 0 a 31. Ils ne sont pas modifiables dans l'editeur. Un IV de 31 est parfait (affiche en or). Ces valeurs sont fixees a la capture/eclosion.

Comment utiliser

Connectez-vous dans l'onglet Editeur avec votre pseudo + mot de passe. Modifiez les EVs/natures de votre equipe, puis cliquez Sauvegarder. Les changements s'appliquent au prochain lancement du jeu.

Guide de l'emulateur

Tout ce qu'il faut savoir pour profiter du jeu sur mobile ou PC.

Sauvegarde simplifiee

Pas besoin de toucher aux states, d'importer ou d'exporter quoi que ce soit. Sauvegardez simplement en jeu (Menu > Sauvegarder) et attendez le message vert de confirmation. Votre progression est automatiquement stockee dans le cloud.

Accelerer le jeu

Appuyez sur les 3 barres en haut a droite de l'emulateur, puis Speed Options > Fast Forward = Enabled. Choisissez le ratio de vitesse souhaite (x2, x4, etc). Attention : le son sera aussi accelere, on conseille de le couper si vous utilisez cette option.

Acces au portail

Un bouton "Portail" est accessible directement depuis l'interface du jeu, au-dessus de l'emulateur. Vous pouvez consulter le Pokedex, les routes et les infos sans quitter la page.

Mode paysage recommande

Sur certains telephones en mode portrait, le pseudo, le bouton Portail et Deconnexion peuvent cacher les 3 barres de menu de l'emulateur. Si c'est le cas, passez en mode paysage (horizontal) pour y acceder.

Plein ecran

Pour une experience immersive, activez le plein ecran via les 3 barres en haut a droite de l'emulateur. Ideal sur mobile pour profiter de tout l'ecran !
